Search and Storage Engine

Xapiand is a fast, simple and modern search and storage engine.

Fast

Xapiand is fast, it relies in the Xapian search engine and is written in modern C++ for efficiency and scalability.

Benchmarks →

Simple

Straightforward RESTful API is JSON document oriented with no need for upfront schema definition.

Quick-start →

Modern

(Near) Real Time Search and data Storage with Multi Tenant & Multi Types indexes also supporting Geospatial queries.

Getting Started →

Get up and running in seconds.

{% highlight sh %} ~ $ brew tap Kronuz/tap ~ $ brew install xapiand ~ $ mkdir my-database ~ $ cd my-database ~/my-database $ xapiand # => Ready to index at http://localhost:8880 ~ $ curl 'localhost:8880/msgs' -d '{ "user": "Kronuz", "message": "Trying out Xapiand..." }' # => Ready to search! ~ $ curl 'localhost:8880/msgs/:search?q=user:Kronuz' {% endhighlight %}
