title: Field Types
---

{{ site.name }} supports a small number of different datatypes for the fields
in a document:

## Concrete datatypes

### String datatypes

* [keyword](../types/keyword-type)
* [text](../types/text-type)

### Numeric datatypes

* `integer`
* `positive`
* `float`

### Date/Time datatypes

* `date`
* `time`
* `timedelta`

### Boolean datatype

* `boolean`

## Geospatial datatypes

* `geospatial`

### UUID datatype

* `uuid`

### Script datatype

* `script`


## Complex datatypes

### Object type
* `object`

### Array type
* `array`

### Foreign type
* `foreign`
